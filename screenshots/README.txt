HOW TO ADD SCREENSHOTS:

1. Run the application (php spark serve)
2. Navigate to each page/report
3. Take screenshots using:
   - Windows: Print Screen or Windows+Shift+S
   - Mac: Command+Shift+4
   - Browser Developer Tools: F12, then device toolbar
4. Save as PNG files in this folder
5. Name them exactly as listed above

✓ User interface design
✓ Data presentation
✓ Report formatting
✓ Filter/search capabilities
✓ Mobile responsiveness (for sales_order)
